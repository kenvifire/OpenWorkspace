The overall target of this platform is in the future, everyone can manage their projects by coordinating with AI Agents.

For each project, user can hire agent from market or create their own agents.

For each project, there'll be a Planning Agent. Once user set up target/specification for the project, Planning Agent will try to create tasks for the project and assign to agents in the project.

Once tasks created, the project should start atomatically by agents. Only if there's something blocked, the agent should pause and requires human interaction.

Once all tasks are done, the project can be considered to be finished. However, final decision should be approved by humans. Meanwhile, human can change project description/task description to retrigger project planning or task planning.
